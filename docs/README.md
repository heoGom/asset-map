# Asset Map Docs

이 디렉터리는 Asset Map 프로젝트와 사용자-AI 협업 과정을 저장소 안에 남기기 위한 문서 공간입니다.

## 문서 구성

- [project-context.md](./project-context.md): 프로젝트 전체 컨텍스트
- [backend.md](./backend.md): backend 기술 스택과 실행 정보
- [front.md](./front.md): frontend 기술 스택과 실행 정보
- [requirements.md](./requirements.md): 요구사항과 범위
- [prompts.md](./prompts.md): 주요 사용자 프롬프트 기록
- [decisions.md](./decisions.md): 의사결정 기록
- [work-log.md](./work-log.md): 작업 일지

## 관리 원칙

- Asset Map에 관한 기획, 요구사항, 프롬프트, 의사결정, 작업 일지는 이 디렉터리에 기록합니다.
- 기능 구현 또는 실행 방식이 변경되면 관련 문서를 함께 업데이트합니다.
- 워크스페이스 공통 규칙은 상위 워크스페이스의 `AGENTS.md`와 `docs/common/` 규칙을 따릅니다.
