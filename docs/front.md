# Asset Map Frontend

## 기술 스택

- Next.js 16.2.6
- TypeScript
- Tailwind CSS
- Recharts
- TanStack React Query
- React Hook Form
- Zod

## 현재 상태

기본 홈 화면만 구성되어 있습니다. 복잡한 화면, API 연동, 상태 관리 구조는 아직 포함하지 않습니다.

## 실행 방법

```bash
cd frontend
npm install
npm run dev
```

기본 실행 주소는 `http://localhost:3000`입니다.

## 검증 방법

```bash
cd frontend
npm run build
```
